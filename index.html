A Rock Paper Scissor game where the player competes against the computer. This project is currently a work in progress and aims to implement features such as random computer moves, score tracking, and an intuitive user interface. Tetris

A recreation of the classic Tetris game using HTML, CSS, and JavaScript. Players control falling blocks to complete horizontal lines. The core mechanics are functional, though there is a bug with collision detection.

Sketch Pad

A simple sketchpad application that allows users to draw on a grid. This project was a fun way to experiment with HTML canvas and JavaScript event handling. It includes features such as color selection, grid resizing, and reset functionality.
